Ideia 1:
One Time PAD
Cifra:
Pegar o tamanho do arquivo, gerar um array com esse tamanho, cada elemento deve
ser um byte pseudoaleatorio, então cada byte do arquivo original sera somado 
com cada elemento desse array. Ao final, o array de chaves(o criado 
aleatoriamente), deve ser escrito em um arquivo chamado chave.
Decifra:
Simplesmente ler o arquivo de chave e o arquivo cifrado para dois arrays, e
fazer a subtração do segundo pelo primeiro.

Ideia 2:
Cifra de Vegenere
Cifra:
Escrever em outro array do tamanho do arquivo a ser cifrado, uma palavra chave,
e depois repetir o conteudo do texto claro. Depois fazer uma conta(subtração ou
soma) byte a byte, e o array resultante tera o arquivo criptografado.
Decifra:
Gerar o mesmo buffer, primeiro decifrando os K primeiros elementos da mensagem
(considerando que K é o tamanho da palavra chave), depois utilizando o texto
claro.
